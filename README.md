## Используемые технологии

- **HTML** — семантическая разметка страницы  
- **CSS** — стилизация, градиенты, абсолютное позиционирование, адаптивные элементы  
- **Flexbox** — выравнивание и компоновка блоков  
- **normalize.css** — сброс стилей для кросс-браузерной совместимости  
- **Статическая вёрстка без фреймворков** — чистый код без использования JavaScript или библиотек

## Описание проекта

Это учебный одностраничный сайт, посвящённый эффективным методам обучения. Проект создан в рамках курса [Яндекс Практикум](https://praktikum.yandex.ru/) и демонстрирует применение HTML, CSS и принципов адаптивной вёрстки.

На странице представлены:
- Основные проблемы традиционного подхода к обучению;
- Интересные факты о мозге и обучении;
- Метод Фейнмана — техника глубокого понимания материала;
- 10 принципов быстрого освоения навыков от Джоша Кауфмана.

Дизайн выполнен в современном минималистичном стиле с использованием градиентов, слоёв и абсолютного позиционирования.
